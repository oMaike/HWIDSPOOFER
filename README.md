## 🔍 Visão Geral
Ferramenta PowerShell para modificação indetectável de identificadores de hardware com técnicas avançadas de anti-análise.

## 🚀 Recursos
- **Spoofing Completo**:
  - MAC Address
  - UUID/BIOS
  - Seriais de Disco
  - CPU/GPU IDs
- **Proteção Anti-Análise**
- **Geração de Valores Realistas**

## 📦 Pré-requisitos
- Windows 10/11
- PowerShell 5.1+
- Execução como Administrador
